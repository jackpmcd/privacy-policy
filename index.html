Privacy Policy

Last updated: 17/04/2025

This Chrome extension does not collect, store, or transmit any personal or sensitive user data.

What the Extension Does

The extension simply checks the content of the currently active web page to detect form fields and automatically fill them with predefined values. All processing is done locally in your browser — no data is sent to any server, nor is any data stored or shared.

Data Collection

  • No personal data collected
  • No tracking or analytics
  • No data shared with third parties

Compliance with Google Policies

This extension complies with Google's User Data Policy for Chrome Extensions. Since it does not handle any personal or sensitive user data, there are no special data obligations required.
